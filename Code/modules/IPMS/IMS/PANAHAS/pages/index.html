Inventory System

The system aims to prevent an unexpected shortage of equipment or supplies during an event or an activity. The inventory is categorized into two, the supplies and the equipment. Supplies are items which can beconsumed while the equipment are items that are not consumed,ratherthey are used and does need repairing whenever it is needed. The inventory is updated each time a supply is added, claimed or depreciated so the supply officer would know the available supplies for use in the following day or week. The supply officer should be the only one who would access the inventory system, thus making him an admin.
